# Black-Paradise/AdminBladeUI

## Introduction

`Black-Paradise/AdminBladeUI` Blade UI for `Black-Paradise/LaravelAdmin`

## Installation

To install the package, run the following command:

```sh
composer require black-paradise/admin-blade-ui
```
